h1. Dotfiles

Jeff Kreeftmeijer's dotfiles.

h2. Installation

Install ["homesick":http://github.com/technicalpickles/homesick]:

bc. gem install homesick

Clone the repo with homesick:

bc. homesick clone jeffkreeftmeijer/dotfiles

And symlink it:

bc. homesick symlink jeffkreeftmeijer/dotfiles

Have fun! ;)
