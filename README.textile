h1. Dotfiles

Jeff Kreeftmeijer's dotfiles.

h2. Installation

Just clone the repository, @cd@ into the @dotfiles@ directory and run @rake install@;

bc. git://github.com/jeffkreeftmeijer/dotfiles.git
cd dotfiles
rake install

This will _remove_ (make backups) your current dotfiles and make symlinks to the files in the dotfiles directory.

Have fun! ;)
